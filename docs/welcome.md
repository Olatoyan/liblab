# Welcome to LibLab

<div className="command-block">
  <h2>Build developer friendly SDKs in minutes from your APIs</h2>

```bash

# Install liblab
npm install -g liblab
# login
liblab login
# Create your config file
liblab init --spec https://petstore.swagger.io/v2/swagger.json
# Generate your SDK
liblab build
```
